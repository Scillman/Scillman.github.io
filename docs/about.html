Jekyll

This is a simple GitHub website generated using Jekyll. Which allows the tutorials to be written completely in Markdown - to be read offline - while providing a staticly generated website for everyone to read them online.

About The Content

The tutorials are all written per version so as to allow modders to see an accurate example of how to make a mod for their respective version of Minecraft. As seemingly a lot of questions pop up where the changes between versions of Minecraft are causing a lot of trouble to new modders.
